# មគ្គុទ្ទេសក៍ការ Hosting គម្រោងជាមួយ Vercel (ល្បឿនលឿន និងឥតគិតថ្លៃ ១០០%)

ឯកសារនេះណែនាំបងពីរបៀបបង្ហោះ (Host) គម្រោង `businessdata168` ទៅកាន់ **Vercel** ដើម្បីឱ្យវេបសាយដំណើរការបាន**លឿនបំផុត (ចុចភ្លាមចេញភ្លាម មិនចាំបាច់រង់ចាំដាស់ម៉ាស៊ីន)** និងអាចភ្ជាប់ឈ្មោះ Domain ផ្ទាល់ខ្លួន (`.com`) បានដោយឥតគិតថ្លៃ។

## 💡 ហេតុអ្វីបានជាជ្រើសរើស Vercel?
- **ល្បឿនលឿនបំផុត**៖ Vercel ប្រើប្រាស់ប្រព័ន្ធ CDN ធ្វើឱ្យវេបសាយបើកភ្លាមចេញភ្លាម គ្មានការចូលគេង (Sleep) ដូច Render ឡើយ។
- **ឥតគិតថ្លៃ ១០០%**៖ គម្រោង Free (Hobby) របស់ Vercel គឺគ្រប់គ្រាន់សម្រាប់គម្រោងរបស់បង។
- **សុវត្ថិភាពទិន្នន័យ**៖ ដោយសារយើងបានភ្ជាប់ Database ទៅកាន់ **Upstash Redis** រួចរាល់ហើយ ដូចនេះទិន្នន័យទាំងអស់នឹងមិនបាត់បង់ឡើយ ទោះបីជា Vercel អាប់ដេតកូដថ្មីក៏ដោយ។

## ជំហានទី ១៖ បង្កើតគម្រោងថ្មីនៅលើ Vercel
1. ចូលទៅកាន់គេហទំព័រ [vercel.com](https://vercel.com)។
2. ចុច **Sign Up** ឬ **Log In** រួចជ្រើសរើសយក **GitHub** ដើម្បីចូលគណនី។
3. នៅលើ Dashboard ចុចលើប៊ូតុង **Add New...** (ខាងស្តាំដៃ) រួចជ្រើសរើសយក **Project**។
4. ត្រង់ផ្នែក **Import Git Repository**៖ ស្វែងរកឈ្មោះ **`businessdata168`** រួចចុចប៊ូតុង **Import**។

## ជំហានទី ៤៖ របៀបភ្ជាប់ទៅកាន់ Domain ផ្ទាល់ខ្លួន `.com`
បន្ទាប់ពីទិញឈ្មោះ Domain ពីក្រុមហ៊ុនលក់ (ឧទាហរណ៍៖ `businessdata168.com`) រួចរាល់ហើយ សូមបងធ្វើតាមវិធីនេះ៖
1. នៅក្នុងគម្រោងរបស់បងលើ Vercel ចូលទៅកាន់ផ្នែក **Settings** (របារខាងលើ) រួចរើសយកផ្នែក **Domains** (ម៉ឺនុយខាងឆ្វេង)។
2. វាយឈ្មោះ Domain របស់បងចូល (ឧទាហរណ៍៖ `businessdata168.com`) រួចចុច **Add**។
3. Vercel នឹងបង្ហាញតម្រូវការកំណត់ DNS Records (A Record ឬ CNAME)។
4. បងគ្រាន់តែចូលទៅកាន់គេហទំព័រដែលបងបានទិញ Domain នោះ (ដូចជា GoDaddy ឬ Namecheap) រួចចូលទៅកន្លែងកំណត់ **DNS Management** ហើយបន្ថែម៖
   - **Type**: `A` | **Name**: `@` | **Value**: `76.76.21.21`
   - **Type**: `CNAME` | **Name**: `www` | **Value**: `cname.vercel-dns.com`
5. រង់ចាំប្រហែល ១០ នាទី នោះវេបសាយរបស់បងនឹងអាចចូលតាមរយៈ `https://businessdata168.com` បានដោយជោគជ័យ និងដើរលឿនភ្លាមៗតែម្តង!
